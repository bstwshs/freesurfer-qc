# QC FreeSurfer — Прототип контроля качества сегментации

## Описание

Веб-приложение на Python для автоматического контроля качества (QC) результатов обработки МРТ-снимков программой FreeSurfer. Сравнивает объёмы подкорковых структур и толщину кортикальных регионов с нормативными значениями (Z-score), выдаёт вердикт OK / Check / Bad, при браке генерирует изображение среза мозга с подсветкой проблемной зоны.

## Стек

- Python 3.9+
- Pandas (обработка таблиц)
- NiBabel (работа с NIfTI)
- Streamlit (интерфейс)
- Matplotlib (графики и визуализация)

## Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone <URL_репозитория>
cd qc_project

# 2. (Опционально) Создать виртуальное окружение
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Сгенерировать тестовые данные
python generate_fake_data.py

# 5. Запустить приложение
streamlit run app.py
```

## Использование

1. После запуска откройте браузер по адресу http://localhost:8501
2. В поле «Путь к папке субъекта» введите `./test_data/subject_001`
3. Нажмите кнопку «Запустить QC»
4. Изучите результаты: таблицу Z-score, гистограмму, распределение вердиктов
5. При наличии регионов с вердиктом Bad — просмотрите срез мозга

```
└── test_data/
    └── subject_001/

        ├── aparc.stats
        └── brain.mgz
```

## Примечания

- Прототип работает с синтетическими данными. Для использования с реальными файлами FreeSurfer замените содержимое `test_data/subject_001/` на реальные файлы.
- При переходе на реальные данные может потребоваться корректировка `parser_engine.py` (форматы реальных .stats файлов могут незначительно отличаться).
